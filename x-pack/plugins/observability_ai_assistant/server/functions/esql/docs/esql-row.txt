ROW
The ROW source command produces a row with one or more columns with
values that you specify. This can be useful for testing.
    ROW a = 1, b = "two", c = null
  a:integer   b:keyword   c:null
  ----------- ----------- --------
  1           "two"       null
Use square brackets to create multi-value columns:
    ROW a = [2, 1]
ROW supports the use of functions:
    ROW a = ROUND(1.23, 0)
