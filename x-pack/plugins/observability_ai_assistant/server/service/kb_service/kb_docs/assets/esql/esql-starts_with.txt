STARTS_WITH
Returns a boolean that indicates whether a keyword string starts with
another string:
    FROM employees
    | KEEP last_name
    | EVAL ln_S = STARTS_WITH(last_name, "B")
  last_name:keyword   ln_S:boolean
  ------------------- --------------
  Awdeh               false
  Azuma               false
  Baek                true
  Bamford             true
  Bernatsky           true
