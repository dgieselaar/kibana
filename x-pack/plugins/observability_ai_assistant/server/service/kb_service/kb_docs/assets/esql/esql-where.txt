WHERE
Use WHERE to produce a table that contains all the rows from the input
table for which the provided condition evaluates to true:
    FROM employees
    | KEEP first_name, last_name, still_hired
    | WHERE still_hired == true
Which, if still_hired is a boolean field, can be simplified to:
    FROM employees
    | KEEP first_name, last_name, still_hired
    | WHERE still_hired
